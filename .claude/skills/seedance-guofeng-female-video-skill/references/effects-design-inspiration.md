# 古风女角色特效层隐性灵感库

这不是给用户直接点选的特效菜单。  
这是给模型做 `特效来源 + 特效主轴 + 强度 + 时机 + 与角色和镜头关系` 联想与收敛的隐藏材料。  
用途不是回答“要什么酷炫特效”，而是回答这些问题：

- 这个角色是否真的需要特效
- 特效是从哪里生出来的
- 特效是在放大角色，还是在抢角色
- 特效该落在开场、中段、爆点，还是只留余韵

## 使用原则

- 把这份文件当成 `特效设计语义地图`
- 不要机械复用原句
- 不要把这里的类别当成法术或元素菜单直接给用户选
- 特效必须绑定来源，不能写成无来源漂浮能量
- 特效必须绑定动作或空间，不能孤立存在
- 特效不能遮脸、毁轮廓、替代动作
- v1 默认只做轻到中度特效，不把古风角色视频推向重游戏 CG

## 1. 环境型特效

### 典型来源

- 风
- 雨
- 雾
- 雪
- 水纹
- 烛火
- 碎光
- 灰烬

### 适合放大的角色

- 清冷
- 空灵
- 悲悯
- 危险但克制的角色

### 适合的视频风格

- 月色空灵感
- 冷雾留白感
- 夜雨压迫感
- 写意水墨感

### 适合的镜头 / 动作关系

- 人不大动，环境替她动
- 镜头先看环境，再补抓她
- 收尾靠残留粒子、风雪、水汽收住

### 容易跑偏的时候

- 环境粒子太密，把人吞掉
- 风雨雾写成纯热闹背景，和角色无关

## 2. 身法型特效

### 典型来源

- 残影
- 拖尾
- 衣摆带出的粒子
- 步伐余光
- 身法切空间时留下的细线或断光

### 适合放大的角色

- 明媚英气
- 危险
- 孤绝天才
- 冷感但有爆点的角色

### 适合的视频风格

- 高级预告
- 动作片爆点推进
- 角色展示里带一点压场感

### 适合的镜头 / 动作关系

- 镜头跟动作走，特效跟着动作轨迹走
- 只在中段推进或高光爆点显出来
- 起点和终点要清楚，不能满屏拖

### 容易跑偏的时候

- 只剩拖尾，没有角色身体语言
- 残影过多，像多角色分身

## 3. 武器型特效

### 典型来源

- 剑气
- 符光
- 刀光
- 金属火星
- 裂纹光

### 适合放大的角色

- 剑修
- 危险角色
- 高位压场角色
- 需要武器记忆点的角色

### 适合的视频风格

- 动作预告
- 冷感压迫式展示片
- 暗黑东方奇幻

### 适合的镜头 / 动作关系

- 武器出手才有特效
- 特效沿武器形制和运动轨迹生成
- 收尾可以停在武器余光而不是满屏爆炸

### 容易跑偏的时候

- 武器特效变成纯光柱，武器本体消失
- 特效强过武器形制，导致角色识别下降

## 4. 术法 / 元素型特效

### 典型来源

- 冰裂
- 火星
- 雷弧
- 寒气
- 幽光

### 适合放大的角色

- 非人感强
- 神性偏强
- 危险奇幻
- 有明确术法设定的角色

### 适合的视频风格

- 暗黑东方奇幻
- 强预告感
- 带术法感的角色片段

### 适合的镜头 / 动作关系

- 术法必须由手势、武器、步伐、视线或空间规则引出
- 最适合用在中段推进或一个明确高光点
- 特效出现后要能回收，不要一直满屏挂着

### 容易跑偏的时候

- 写成游戏 CG 的全屏法术爆炸
- 颜色、元素、粒子堆太满，压掉角色和场景风格

## 5. 收尾型特效

### 典型来源

- 余光
- 残粒
- 散去
- 熄灭
- 回收

### 适合放大的角色

- 清冷
- 贵气
- 破碎
- 克制但危险的角色

### 适合的视频风格

- 电影片段
- 海报延展
- 情绪片
- 高级角色预告

### 适合的镜头 / 动作关系

- 角色先成立，特效最后留下余韵
- 特效不必爆开，只要把记忆点留住

### 容易跑偏的时候

- 收尾特效比收尾画面还重要
- 角色已经离场，特效还在无意义地飘

## 6. 强度判断

- `轻`：只作为气氛、边缘、余韵，优先环境型或收尾型
- `中`：能看到来源和轨迹，但仍以角色和动作为主
- `偏强`：只用于一个明确爆点，且必须有主轴理由

默认不要超过 `偏强`。  
如果用户要求重游戏 CG，要先按冲突处理逻辑处理，不要直接迎合。

## 7. 特效层自检

如果特效设计还不稳，回头检查：

- 特效是不是有明确来源
- 特效是不是有明确出现时机
- 特效是不是在放大角色或镜头，而不是替代它们
- 特效有没有盖住脸、武器、本体轮廓
- 如果把特效拿掉，这条视频是否仍然成立

如果最后一个问题答案是“完全不成立”，说明特效已经抢戏了。
