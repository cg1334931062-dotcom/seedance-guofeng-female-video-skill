# 提示词质量测试用例

## 测试标准
所有生成的提示词必须符合以下标准：
1. ✅ 结构清晰，要素完整
2. ✅ 没有模糊描述（如"好看"、"漂亮"等）
3. ✅ 符合Seedance/SeedDream最佳实践
4. ✅ 所有用户确认的要素都准确体现
5. ✅ 不同工作模式的输出格式正确

---

## 测试用例1：仅视频模式提示词质量
**输入**：
```
角色：清冷孤绝的女剑仙，20岁左右，白衣胜雪，手持流云剑，眉心有一点红痣
场景：万米高空的云海之上，金色朝霞洒下，角色独立于剑尖
风格：电影质感，冷色调，高对比度，构图留白多
时长：8秒
镜头：从全景慢慢推近到面部特写，最后定格在眉心红痣
动作：衣袂飘飘，发丝飞舞，眼神淡漠看向远方，右手轻轻抬起，指尖有微弱剑光
特效：无
```

**预期输出**：
```
# 清冷女剑仙 Seedance 视频提示词

## 角色脸谱摘要
20岁清冷女剑仙，白衣，持流云剑，眉心红痣，气质孤绝

## 一句话角色定义
白衣胜雪的清冷女剑仙独立于云海剑尖，眉心红痣是标志性记忆点

## 角色设计摘要

## 视频风格摘要
美学方向：电影质感，色彩：冷色调高对比度，手法：大量留白，情绪基调：孤寂淡漠

## 镜头设计摘要
运镜逻辑：全景→中景→近景→面部特写，总时长：8秒，传播节奏：慢节奏，适合意境表达，入口：云海全景，角色在远处只有一个小点，推进：慢慢推近，逐渐看清服装、武器、面部特征，收尾：定格在眉心红痣特写

## 动作设计摘要
动作母题：独立远眺，身体语言：衣袂飘飘，发丝飞舞，右手轻抬，力量表达：指尖有微弱剑光，内敛不张扬

## 质检与修正建议
✅ 时长匹配度：8秒视频4个镜头阶段，节奏舒缓，匹配良好
✅ 信息密度：所有核心要素完整，细节丰富
✅ 特效风险：无特效，不会抢戏
✅ 提示词清晰度：没有模糊描述，所有细节明确
💡 参考图策略：建议先生成角色立绘确认形象，再用参考图生成视频，保证角色一致性
```

## 测试用例4：最佳实践融入验证
**要求**：
- 提示词自动包含Seedance官方推荐的关键词
- 避免官方明确禁止的描述
- 结构符合官方推荐的顺序：主体→场景→动作→镜头→风格

**正面检查（必须包含）**：
- [x] 明确的时长说明（如"8秒视频"）
- [x] 明确的镜头运镜说明
- [x] 动作的时间节点描述
- [x] 风格关键词明确
- [x] 分辨率/质量说明（如"8K高清"）

**负面检查（禁止出现）**：
- [ ] "最好看的"、"最漂亮的"等主观描述
- [ ] "两只手"、"五个手指"等数量描述
- [ ] 没有明确含义的网络热词
- [ ] 自相矛盾的描述（如"恐怖的可爱"）

## 测试用例5：质检报告准确性
**输入场景**：
```
角色：温柔娴静的大家闺秀
场景：战场上的烽火之中
动作：激烈打斗，舞刀弄枪
时长：5秒
镜头：全景→中景→特写→跳切→收尾，共5个镜头切换
特效：全身覆盖高强度的火焰特效，遮挡住面部
```

**预期质检报告**：
```
## 质量检查报告
❌ 检查未通过：存在以下必须修正的问题
- 角色设定与动作冲突："温柔娴静"的气质与"激烈打斗"的动作不匹配
- 时长与镜头不匹配：5秒视频有5个镜头切换，节奏过快，观众无法看清内容
- 特效风险过高：高强度火焰特效遮挡面部，会导致角色辨识度极低

⚠️ 警告信息
- 角色与场景冲突："大家闺秀"的身份与"战场"场景风格冲突

💡 优化建议
- 建议调整角色气质或动作设计，保持风格统一
- 建议将镜头减少到2个以内，保证5秒内可以清晰展示
- 建议降低特效强度，或者调整特效位置避免遮挡面部
- 建议调整场景设定，使其与角色身份匹配
```
