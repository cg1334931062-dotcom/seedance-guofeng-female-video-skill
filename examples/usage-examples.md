# Usage Examples

## 1. 默认进入仅视频

用户输入：

```text
$seedance国风女性视频skill 帮我做一个清冷女剑修视频
```

预期行为：

- 默认进入 `仅视频`
- 不误判成立绘
- 先锁角色和场景，再进入视频风格、总时长、镜头、动作

## 2. 明确讨论总时长

用户输入：

```text
$seedance国风女性视频skill 我想做一个带预告感的古风女角色视频
```

预期行为：

- 在镜头层必须问 `视频总时长`
- 优先给出 `5-8 秒 / 8-12 秒 / 12-18 秒` 档位
- 没确认时长前不进入最终收敛

## 3. 用户直接给秒数

用户输入：

```text
$seedance国风女性视频skill 做一个 8 秒的女剑修 Seedance 视频
```

预期行为：

- 直接接受 `8 秒` 作为已确认总时长
- 镜头和动作密度必须服从这个时长

## 4. 整链路模式

用户输入：

```text
$seedance国风女性视频skill 先帮我做角色稿，再做视频
```

预期行为：

- 进入 `整链路`
- 先稳定角色稿
- 再收敛视频风格、总时长、镜头、动作

## 5. 仅角色稿模式

用户输入：

```text
$seedance国风女性视频skill 帮我做一张古风女角色立绘海报
```

预期行为：

- 进入 `仅角色稿`
- 不强行打开视频流程

## 6. 六项收敛回显

对视频任务，最终输出前应明确回显并确认：

- 角色
- 场景
- 视频风格
- 视频总时长
- 镜头设计
- 动作设计
