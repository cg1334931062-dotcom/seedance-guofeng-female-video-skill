# 月雾感偷闲小师妹 Seedance 仅视频成品示例

角色脸谱摘要
- 角色身份：仙门中的偷闲小师妹
- 年龄脸感：18-20 岁的年轻少女脸感，圆润鹅蛋脸，眼神半醒半梦
- 气质：软糯呆懵，慵懒松弛，带一层月雾感轻灵
- 发型：松散双髻，几缕碎发自然垂落
- 妆容：干净淡妆，肤感透亮，浅粉唇色，眼周克制不浓艳
- 服装与武器：浅青与月白配色的轻纱襦裙、宽袖外衫，无武器
- 色彩主轴：冷白青蓝
- 角色记忆点：一枚微微歪着的小银铃

一句话角色定义
- 像被晨雾养大的仙门小师妹，偷闲犯困时也带着不沾尘的轻灵。

创意收敛说明
- 这支视频优先放大的不是“仙门设定”本身，而是她那种慵懒、松弛、软糯，却又被晨雾和水气托得很轻的偷闲感。

角色设计摘要
- 角色：不是高冷女主，也不是活泼俏皮挂，而是松弛、慢半拍、让人一眼记住的偷闲小师妹
- 脸谱稳定点：年轻少女脸感，圆润但不幼，半醒半梦的眼神，松散双髻
- 服装与武器：浅青月白轻纱襦裙和宽袖外衫，不配武器，避免动作主轴被打斗感带偏
- 最不能丢的记忆点：歪着的小银铃和她那种“刚醒又没完全醒”的回眸状态

场景设计摘要
- 核心场景：清晨薄雾中的莲池水阁
- 角色与场景关系：她不是摆拍站在景里，而是自然倚在窗边或案几旁偷闲犯困
- 环境压迫 / 释放：环境不压她，薄纱、木栏、池水、荷叶和晨雾一起托住她的松弛感
- 光线 / 天气质感：清晨冷白青蓝调，轻雾、柔光、微潮水气
- 构图锚点：薄纱或荷叶前景，窗边人物上半身，最后落到回眸余韵和铃铛细节

视频风格摘要
- 渲染基底：仙侠过场 CG
- 画面质感：材质干净精修，带稳定的仙侠游戏过场质感
- 虚实关系 / 空气感：人清景雾，人物脸和服装轮廓清楚，环境带轻雾和柔光
- 情绪主轴：轻、慢、透，带一点犯困时的柔软游离
- 成片感觉：像高质量仙侠游戏剧情过场中的一小段角色氛围片段
- 为什么适合这个角色：这种风格能稳住脸、发型和服装轮廓，同时保留晨雾和水阁的轻灵空间感，不会把她做成水墨虚影或海报摆拍

镜头设计摘要
- 视频总时长：10 秒
- 传播节奏：标准推进，轻慢呼吸感
- 视频类型：单角色氛围展示片
- 入口镜头：隔着被晨风轻轻吹动的薄纱或荷叶进入画面，先看到水阁窗边犯困的小师妹
- 情绪推进：镜头缓慢推近，从整体空间关系推进到人物上半身和脸
- 空间关系：前景薄纱或荷叶，中景人物和木栏，后景池水与晨雾
- 收尾方式：停在她慢伸懒腰后回眸的余韵，以及小银铃轻轻晃动的细节

动作设计摘要
- 动作母题：慢伸懒腰后回眸
- 身体语言：肩颈先松开，宽袖和手腕带出轻软拖曳，再慢慢转头回眸
- 力量表达：没有武打爆发，只有很轻的舒展、回带和停顿
- 动作与气质关系：动作要服务她的慵懒轻灵，不做夸张萌态，不做热血表演
- 动作密度与时长关系：10 秒里只保留一个入口、一个推进、一个收尾，不再叠加第二个主动作
- 参考图策略：纯文本直出，用文字重复角色稳定点

```text
整支视频保持轻、慢、有呼吸感的标准推进，不要快切，不要多人，不要武打，不要大幅度动作，不要强特效，不要水墨虚化，不要高海报压场。始终保持同一个角色、同一张脸、同一发型、同一套浅青月白轻纱服装和同一枚小银铃，重点是她慵懒轻灵的偷闲感。
```

一致性增强提示：
```text
只保留这一个角色，不生成第二个人物。稳定保持仙门偷闲小师妹的身份，18-20 岁年轻少女脸感，圆润鹅蛋脸，半醒半梦的眼神，松散双髻，浅青月白轻纱襦裙和宽袖外衫，歪着的小银铃，无武器。风格固定为仙侠过场 CG，冷白青蓝调，人物脸和服装轮廓清楚，环境只有轻雾和柔光，不要把人物一起雾化，不要变成水墨风，不要变成宣传海报感，不要暗黑奇幻，不要夸张游戏特效。镜头固定为 10 秒内的一个入口、一个缓慢推进、一个余韵收尾，动作固定为轻小的伸懒腰和慢回眸。
```

质检与修正建议
- 先试 `主提示词`，如果人物脸感不稳，再叠加 `一致性增强提示`
- 如果第一轮角色变成泛古风少女，优先强化“偷闲小师妹 + 松散双髻 + 歪着的小银铃”这三个稳定点
- 如果第一轮画面太梦太糊，优先补强“人清景雾、人物脸和服装轮廓清楚、仙侠过场 CG”这几个风格句
- 如果第一轮镜头像模板，优先重写入口镜头里的“薄纱或荷叶前景带入”以及人物和窗边的空间关系
- 如果第一轮动作太大或太活泼，优先把动作收回“肩颈舒展、手腕拖曳、慢回眸”，删掉额外表演
- 如果第一轮铃铛不明显，优先把收尾句再写重一点，但不要同时增加新配饰
- 如果第一轮出现多人、武器或强特效，先删这些干扰项，不要继续堆更多风格词
