# 古风女角色场景层隐性灵感库

这不是给用户直接点选的场景菜单。  
这是给模型做 `角色与空间关系 + 环境动力 + 光线时间 + 构图锚点` 联想和差异化生成的隐藏材料。  
用途不是回答“她在哪”，而是回答这些问题：

- 这个角色在什么空间里最成立
- 空间是在压她、托她、藏她，还是被她夺回
- 环境里什么在动，什么在替她说话
- 画面应该先让观众看到什么，再意识到什么

## 使用原则

- 把这份文件当成 `场景设计语义地图`
- 不要机械复用原句
- 不要把这里的关系主轴当成固定菜单逐条给用户选
- 先想角色和空间是什么关系，再想地点名词
- 场景不该只是背景板，而要直接放大角色脸谱和视频逻辑

## 1. 被空间压住

### 场景关系

- 空间先成立，人物像被规则困在里面
- 观众先感到冷、重、空、压，再注意到她

### 环境动力

- 雾、回廊、空殿、冷雨、门规感很强的秩序空间
- 环境先发声，人物后出现

### 光线与时间

- 夜雨
- 冷雾清晨
- 低照度烛光

### 适合放大的角色

- 孤绝天才
- 破碎
- 被命运压着走的人
- 冷得不近人情的人

## 2. 由空间托起

### 场景关系

- 空间像在给她让路
- 不是她被景吞掉，而是景把她托到最强的位置

### 环境动力

- 云海
- 月色水面
- 山门高处
- 风从她身后或两侧打开空间

### 光线与时间

- 月夜
- 清晨逆光
- 透亮但不过曝的冷光

### 适合放大的角色

- 空灵
- 贵气
- 清冷
- 神性偏强的角色

## 3. 由她夺回空间

### 场景关系

- 空间原本有压迫或混乱感，但她一进入就改写了画面秩序
- 环境不是先天属于她，是被她压住或切开的

### 环境动力

- 夜雨长街
- 火光压场的旧殿
- 风雪中的高台
- 带反光地面的狭长通道

### 光线与时间

- 夜色反光
- 冷黑金属感火光
- 有明显方向性的逆光

### 适合放大的角色

- 危险
- 明媚英气
- 孤绝天才
- 克制但有爆点的角色

## 4. 藏于空间

### 场景关系

- 第一次不完全看清她
- 角色像被风、雾、纱、屏风、雨帘先遮住，再被补抓出来

### 环境动力

- 前景遮挡
- 水汽
- 帘幕
- 树影
- 局部烛火

### 光线与时间

- 月下薄雾
- 微雨傍晚
- 柔光梦境感

### 适合放大的角色

- 神秘
- 妖异
- 清冷
- 悲悯

## 5. 环境动力库

这些不是地点菜单，而是环境如何替角色说话：

- 风：放大发丝、衣摆、孤高感、空灵感
- 雨：增加压迫、危险、反光和重量
- 雪：放大冷、距离、命运感、易碎感
- 雾：放大留白、不完整性、神秘感
- 烛光：放大贵气、旧事感、禁欲克制
- 水面 / 水汽：放大倒影、双重情绪、月色流动
- 回廊 / 门框 / 殿柱：强化秩序感、压迫感、构图稳定性
- 山门 / 高处 / 屋檐边缘：强化高位感、孤绝感、世界关系
- 城巷 / 暗巷：强化危险、现实感、反差感

## 6. 光线与时间质感

- 月夜：冷、透、远，适合空灵、清冷、神性
- 晨光：轻、开、希望中带距离，适合明媚或被托起的角色
- 逆光：适合先显轮廓，再显脸
- 冷雾：适合悲悯、破碎、未说出口的前史
- 湿冷反光：适合危险、压迫、冲突前夕
- 火光压场：适合贵气、危险、旧殿秩序感

## 7. 构图锚点

- 入口视线落点：观众第一眼抓哪里，是眼神、手、武器、侧脸，还是被风带起的衣摆
- 前景遮挡：用纱、帘、树影、柱子、水汽增加“先不完全看清”
- 空镜释放：在补抓到角色之前，允许空间先说话
- 收尾定格位：最后停在脸、手、武器、背影，还是角色与空间的关系上
- 纵深方向：长廊、台阶、桥面、水岸都可以帮助镜头推进

## 8. 反差关系

这些关系更适合在用户方向太顺、太稳、太普通时使用：

- 贵气角色进入破败冷巷
- 危险角色进入圣洁明净空间
- 破碎角色置于过亮、过干净的空间
- 清冷角色放进有烟火气的旧城巷
- 空灵角色放进湿冷夜雨场景，做出不属于那里的反差

## 9. 场景层自检

如果场景设计仍然像背景板，回头检查：

- 能不能一句话说清角色和空间是什么关系
- 环境里什么在替她动
- 光线和天气有没有真的参与气质表达
- 入口镜头和收尾画面是不是都能落到场景锚点上
- 如果去掉这个场景，角色会不会几乎不受影响

如果最后一个问题答案是“会”，说明场景还没设计到位。
