# SeedDream Best Practices

## Core Principle

Use concise natural language, not keyword piles.

Preferred shape:

```text
[图片类型/用途]。[主体是谁，处于什么状态或姿态]。[环境/背景]。[风格、色彩、光影、构图要求]。
```

## What to Lock Early

For role-sheet or character design prompts, lock these first:

- 主体身份
- 气质
- 风格方向
- 发型 / 妆容
- 服装结构
- 武器
- 背景
- 用途：角色设定图 / 角色立绘 / 海报

## Role-Sheet Guidance

- State the image type explicitly.
- Keep the silhouette readable.
- Prefer stable full-body or clear half-body composition when the image may later be used as a reference.
- Say what must remain clear: face, hair, costume palette, costume structure, weapon shape.

## Reference-Image Guidance

When the user already has a reference image or wants one later, prefer language that preserves:

- 脸部特征
- 发型
- 服装配色与结构
- 武器形制
- 整体气质

## Avoid

- long adjective waterfalls
- disconnected tags
- vague phrases like “高级一点” without follow-up clarification
