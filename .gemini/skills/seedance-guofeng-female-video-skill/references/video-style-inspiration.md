# 视频风格隐性灵感库

这不是给用户直接点选的风格菜单。
这是给模型把“导演式对话”映射成可执行风格决策的隐藏材料。

新的默认结构不是让用户显式共创五个风格字段，而是：

- 用户侧只共创两个子层：
  - `美术风格基底`
  - `成片风格 / 导演表达`
- 模型内部再落到三个执行轴：
  - `美术基底`
  - `成片对象`
  - `画面质地与情绪边界`

`为什么适合当前角色与场景` 继续保留，但应主要用于回显和最终输出，而不是单独再开一问。

## 使用原则

- 把这份文件当成 `导演沟通 -> 执行风格` 的映射表
- 风格层默认在用户侧 `3 问内锁定`
- 用户侧优先用“像哪类片”“画面更怎么呈现”来问，不先平铺术语
- 如果需要暴露术语，最多自然化地使用 `美术基底`、`成片类型`
- 不要把这里的分类原样变成固定选项菜单
- 先按当前角色与场景合成方向，再用本文件检查方向是否够清楚、够不冲突

## 用户侧默认顺序

### 第 1 问：美术风格基底

先问这支片的画面更像什么被做出来：

- `仙侠过场 CG`
- `宣传主视觉 CG`
- `国风动漫片段`
- `写意水墨`
- `插画动效`
- `柔光拟实拍记忆片段`

### 第 2 问：成片风格 / 导演表达

再问这支片更像什么样的 finished piece：

- `剧情过场`
- `角色预告 / PV`
- `人物片段`
- `梦境片段 / 梦境凝视`
- `情绪碎片`
- `海报延展`

### 第 3 问：画面质地与情绪边界

最后问画面更应该落在哪个边界上：

- `更清` 还是 `更雾`
- `更柔` 还是 `更压`
- `更克制` 还是 `更包装`
- `更轻` 还是 `更湿冷`

这里不要求用户再回答抽象的“情绪主轴 / 空气感”字段，而是让模型把这些自然语言落回内部执行轴。

## 宽词映射规则

### 如果用户说 `游戏CG风格`

不要继续追问抽象术语，优先拆成：

- 是 `仙侠过场 CG` 还是 `宣传主视觉 CG`
- 更像 `剧情过场` 还是 `角色预告 / PV`
- 画面要更 `清`、更 `雾`、更 `克制` 还是更 `包装`

### 如果用户说 `电影感`

优先拆成：

- 更像 `国风动漫片段` 还是 `柔光拟实拍记忆片段`
- 更像 `人物片段 / 剧情过场` 还是 `情绪碎片`
- 画面边界更 `清`、`柔`、`压` 还是 `雾`

### 如果用户说 `水墨感`

优先拆成：

- 是真正的 `写意水墨`，还是只是想要 `轻雾留白`
- 更像 `梦境片段` 还是 `情绪碎片`
- 是否接受画面明显更虚、动作明显更降密度

### 如果用户说 `高级`、`氛围感`、`有感觉`

优先拆成：

- 这支片更像哪类 finished piece
- 画面更应该往哪种边界靠
- 是靠 `基底` 变高级，还是靠 `收束和克制` 变高级

## 导演式风格包

下面这些不是给用户直接照抄的模板，而是给模型做动态合成的高阶参考。

### 1. 仙侠过场剧情片段

- 适合的角色 / 场景语境：
  - 角色和空间都要成立
  - 希望既稳角色脸谱，又保留环境气息
  - 适合清冷、空灵、柔弱、贵气但不重包装的人物
- 用户侧推荐问法：
  - “这支片更像一段游戏里的真实剧情片段，还是更像角色宣发短片？”
  - “你要的是更干净稳定的仙侠过场质感，还是更像强包装角色 PV？”
- 内部 3 轴落点：
  - `美术基底`：仙侠过场 CG
  - `成片对象`：剧情过场 / 人物片段
  - `画面质地与情绪边界`：常见落点是 `人清景雾`、轻克制、偏透气
- 常见冲突与降级策略：
  - 如果用户同时要重 PV 包装，先问谁是主轴
  - 如果用户要很强特效，优先提醒容易冲成宣发感

### 2. 主视觉角色预告

- 适合的角色 / 场景语境：
  - 角色记忆点强
  - 压场、危险、贵气、高位感明显
  - 更重人物包装，不重生活流
- 用户侧推荐问法：
  - “你更想让这支片像角色预告，还是像剧情里截出来的一段？”
  - “是要构图压场的角色包装，还是要真实进入场景的片段感？”
- 内部 3 轴落点：
  - `美术基底`：宣传主视觉 CG
  - `成片对象`：角色预告 / PV
  - `画面质地与情绪边界`：更清、更强构图、更包装
- 常见冲突与降级策略：
  - 和生活流、偷闲感、小人物片段容易冲突
  - 如果用户又要“克制”，把包装感降成辅助轴

### 3. 国风动漫人物片段

- 适合的角色 / 场景语境：
  - 既想要片段感，又不想太游戏
  - 希望人物和空间同等成立
  - 适合轻命运感、清冷、空灵、柔和后劲类角色
- 用户侧推荐问法：
  - “这支片更像国漫电影里的一个人物片段，还是更像游戏过场？”
  - “你希望它更像完整片子里截出的一段，还是更像独立角色展示？”
- 内部 3 轴落点：
  - `美术基底`：国风动漫片段
  - `成片对象`：人物片段 / 剧情过场
  - `画面质地与情绪边界`：常见落点是清而不硬、柔而不糊、后劲克制
- 常见冲突与降级策略：
  - 如果用户再叠强宣发包装，先定主轴再保留次轴

### 4. 写意水墨梦境片段

- 适合的角色 / 场景语境：
  - 空灵、悲悯、破碎、病气、诗意克制
  - 画面允许不那么实，允许边界留白
  - 更像梦境或情绪残片
- 用户侧推荐问法：
  - “你要的真的是水墨成片，还是只是想要轻雾留白？”
  - “这支片更像梦里看见的一瞬，还是像剧情里真实发生的一段？”
- 内部 3 轴落点：
  - `美术基底`：写意水墨
  - `成片对象`：梦境片段 / 情绪碎片
  - `画面质地与情绪边界`：更柔、更透、更留白，动作和特效都应降密度
- 常见冲突与降级策略：
  - 和强游戏 CG、强爆点动作、重包装 PV 最容易冲突
  - 若用户坚持高燃，只能保水墨作辅助气味，不能双主轴平均混写

### 5. 插画动效海报延展

- 适合的角色 / 场景语境：
  - 视觉记忆点强
  - 更看重构图停帧和海报感
  - 空间真实推进不重要
- 用户侧推荐问法：
  - “你想让它更像一张会轻轻动起来的主视觉，还是一段真正推进的片段？”
  - “你更想要强构图的视觉记忆，还是更真实的空间推进？”
- 内部 3 轴落点：
  - `美术基底`：插画动效
  - `成片对象`：海报延展
  - `画面质地与情绪边界`：更设计化、更凝视、更强构图
- 常见冲突与降级策略：
  - 和复杂剧情过场、强镜头推进冲突
  - 如果用户又要真实片段感，优先问哪边是主轴

### 6. 柔光拟实拍记忆片段

- 适合的角色 / 场景语境：
  - 少女感、月色感、记忆感、温柔偷闲感
  - 希望更像摄影机看见的一瞬，而不是完整 CG 片段
- 用户侧推荐问法：
  - “你想让它更像动画片段，还是更像柔光摄影里截出的一小段记忆？”
  - “你要的梦感是更像真实摄影里的柔光，还是更像水墨留白？”
- 内部 3 轴落点：
  - `美术基底`：柔光拟实拍记忆片段
  - `成片对象`：人物片段 / 梦境片段
  - `画面质地与情绪边界`：更柔、更轻、更近记忆感，但要防止人物发糊
- 常见冲突与降级策略：
  - 如果用户又要求角色轮廓极稳，要把“柔”压回可控范围

## 常见冲突

- `写意水墨` + `重游戏 PV 包装`
- `海报延展` + `强剧情过场`
- `柔光拟实拍记忆片段` + `要求极稳人脸轮廓`
- `仙侠过场剧情片段` + `高密度特效爆点`
- `国风动漫人物片段` + `过重宣发包装`

遇到冲突时，不要平均混写。
先定 `美术基底` 或 `成片对象` 谁是主轴，再把 `画面质地与情绪边界` 调到服务主轴的程度。

## 快速自检

在离开风格层前，默默检查：

- 用户侧是否已经在 3 问内锁住风格主轴
- 用户有没有回答出“像哪类片”和“画面更怎么呈现”
- 当前风格是否仍然像抽象术语堆，而不是具体成片
- 内部 3 轴是否已经能支持后面的镜头和动作设计
- `为什么适合当前角色与场景` 是否已经能在 recap 里说清
