# 视频风格隐性灵感库

这不是给用户直接点选的风格菜单。
这是给模型做 `渲染基底 + 虚实关系 / 空气感 + 情绪主轴 + 成片对象 + 适配理由` 分层收敛的隐藏材料。
用途不是给一个笼统“风格名”，而是回答这些问题：

- 这支视频首先属于什么视觉基底
- 人物和环境是清、雾、柔还是湿冷反光
- 观看时的情绪主轴是什么
- 它更像哪一种成片对象
- 为什么这种风格最适合当前角色和场景

## 使用原则

- 把这份文件当成 `视频风格多轴语义地图`
- 风格层默认按 `渲染基底 -> 虚实关系 / 空气感 -> 情绪主轴 -> 成片对象 -> 适配理由` 收敛
- 不要机械复用原句
- 不要直接把这些标题当成给用户的固定菜单
- 如果需要给用户选项，先让模型按当前项目语境合成方向，再用本文件检查这些方向是否够开、够准、够不冲突
- 用它来帮助模型把“高级、仙气、电影感、游戏CG风格、水墨感”这种大词拆成可执行的风格决策

## 1. 渲染基底

### 仙侠过场 CG

- 画面质感：人物清楚，材质干净精修，布料、水面、雾气都有稳定的仙侠游戏过场感
- 最适合放大：空灵、清冷、轻梦、贵气但不重压的角色
- 不适合：极简留白到几乎无材质的片子，或强烈实验影像感
- 常见冲突：容易被写成重游戏特效片或宣发 PV

### 宣传主视觉 CG

- 画面质感：构图更强，包装更重，人物更像角色 PV 或宣发短片里的核心镜头
- 最适合放大：贵气、压场、危险、角色记忆点很强的人
- 不适合：生活流、偷闲感、太软太散的角色
- 常见冲突：容易和“克制展示片”打架，也容易把镜头层带成海报延展

### 国风动漫电影感

- 画面质感：清晰但不过锐，人物与环境都成立，像国漫电影里的一段片段
- 最适合放大：清冷、孤绝、贵气、可感的空灵角色
- 不适合：过于搞怪、过热过躁、只有动作标签没有气场的人
- 常见冲突：容易被用户用一个“电影感”混过去，结果没有锁清到底是电影片段还是角色预告

### 写意水墨

- 画面质感：更虚、更透、边界不是每时每刻都完整给出
- 最适合放大：空灵、悲悯、病气、破碎、诗意克制的人
- 不适合：爆点动作、强招式展示、重游戏 CG 特效
- 常见冲突：最容易和“强特效游戏 CG”“高密度爽感混剪”冲突

### 插画动效感

- 画面质感：像高质量国风插画被轻轻激活，动势比真实空间更受构图控制
- 最适合放大：海报感强、视觉记忆点强、适合做角色定格式延展的人
- 不适合：需要强空间真实感和复杂镜头推进的剧情片段
- 常见冲突：容易和“剧情过场”冲突，因为画面会更像被设计过的动态图

### 柔光梦境感实拍 / 拟实拍

- 画面质感：更像真实摄影或拟实拍镜头，光更柔，边界更软，梦感比硬材质更重要
- 最适合放大：温柔、轻飘、带不真实感的少女气或月色人物
- 不适合：需要锋利轮廓、重压感、重武器形制的角色
- 常见冲突：容易把人物也一起做糊，最后丢脸感和服装轮廓

## 2. 虚实关系 / 空气感

### 人清景雾

- 人物脸和服装轮廓清楚，环境带轻雾、柔光、空气层
- 最稳，特别适合 `仙侠过场 CG`、`国风动漫电影感`
- 适合想保角色稳定又想保空间气氛的时候

### 人景都清

- 人和景都更清楚，空间真实感和材质读感更强
- 适合 `宣传主视觉 CG`、更正式的 `剧情过场`
- 风险是梦感和留白感会下降

### 人景都柔

- 人物和环境都更软，梦感、月感、记忆感更强
- 适合 `写意水墨`、`柔光梦境感实拍 / 拟实拍`
- 风险是人物稳定性下降，容易丢脸感和轮廓

### 湿冷反光

- 空气里有水气和重量，地面或材质带反光
- 适合危险、孤绝、压迫、夜雨一类主轴
- 风险是和轻梦少女气或过甜角色打架

### 柔光薄雾

- 光线柔，雾气轻，氛围比物理重量更重要
- 适合温柔空灵、月色、偷闲感、梦境凝视
- 风险是如果不强调人物清晰度，容易整体发糊

## 3. 情绪主轴

- `冷`：干净、远、克制，不等于无情绪
- `压`：空气里有重量，角色和空间之间有逼迫感
- `梦`：轻、漂、远，像现实和幻觉之间
- `贵`：不是堆料，是人物压得住画面
- `柔`：温柔、松开、靠近，但不一定甜
- `危险`：安静里带刀，或带不安全感
- `神性`：不像完全属于人间，带非凡或非人质感
- `后劲`：当下不爆，但收尾会留住余韵

这些词不是单独风格结论，只是风格层里的一个轴。
不要让 `冷`、`梦`、`危险` 代替 `渲染基底` 或 `成片对象`。

## 4. 成片对象

### 剧情过场

- 更像游戏或影视中的一小段剧情片段
- 需要角色和空间都成立，镜头推进要自然

### 角色预告 / PV

- 更强调角色包装感和记忆点
- 适合 `宣传主视觉 CG` 或 `国风动漫电影感`

### 海报延展

- 每一帧都更像可停下来的强构图画面
- 适合 `插画动效感` 或主视觉式 CG

### 梦境凝视

- 更像一段被拉长的注视或梦中看见的画面
- 适合 `写意水墨`、`柔光梦境感实拍 / 拟实拍`

### 情绪碎片

- 更像一段未说完的前史、回忆或情绪残片
- 适合留白、破碎、悲悯、克制方向

## 5. 常见宽词拆解

### 如果用户说 `游戏CG风格`

优先继续拆：

- 是 `仙侠过场 CG`
- 还是 `宣传主视觉 CG`
- 还是偏轻的二次元 CG
- 人物和环境要 `人清景雾`、`人景都清` 还是 `人景都柔`
- 它更像 `剧情过场` 还是 `角色 PV`

### 如果用户说 `电影感`

优先继续拆：

- 更像 `国风动漫电影感` 还是 `柔光梦境感实拍 / 拟实拍`
- 这是一段 `剧情过场`、`人物片段` 还是 `海报延展`
- 它要更冷、更湿、更锋利，还是更柔、更透、更远

### 如果用户说 `水墨感`

优先继续拆：

- 是 `写意水墨` 还是只是想要轻雾留白
- 人物要不要保持清楚轮廓
- 是否接受动作和特效都被明显降密度

## 6. 常见风格冲突

- `写意水墨` + `强特效游戏 CG`
- `克制展示片` + `重游戏 CG 宣发包装`
- `梦境凝视` + `高燃混剪打斗`
- `人景都柔` + `要求极稳的人物轮廓`
- `海报延展` + `强剧情过场`

遇到冲突时，不要平均混写。
先定主轴，再把次轴降到服务主轴的程度。

## 7. 可直接复用的风格拆句

- `仙侠过场 CG + 人清景雾 + 轻梦 + 剧情过场`
- `宣传主视觉 CG + 人景都清 + 贵压 + 角色 PV`
- `国风动漫电影感 + 柔光薄雾 + 后劲 + 人物片段`
- `写意水墨 + 人景都柔 + 留白 + 梦境凝视`
- `插画动效感 + 人景都清 + 凝视 + 海报延展`

这些不是给用户直接点选的固定答案。
用途是帮助模型快速判断：当前风格已经足够可执行，还是还停留在大词层。
